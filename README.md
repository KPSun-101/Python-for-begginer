# Basic Python Programs

This repository contains basic Python programs for beginners. It covers fundamental programming concepts and provides simple examples to help learn Python step by step.

## Topics

- Variables and Data Types
- Input and Output
- Operators
- Conditional Statements
- Loops
- Functions
- Lists, Tuples, Sets, and Dictionaries
- Strings
- File Handling
- Exception Handling
- Basic Object-Oriented Programming
